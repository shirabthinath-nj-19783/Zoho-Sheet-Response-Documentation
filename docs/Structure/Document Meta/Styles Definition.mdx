# Styles Definition

:::note
Operation type will be changed based on the operation. Possible operationTypes are : META, APPEND, SYNC.
:::

For <b>SYNC</b> operation type only the response structure will be changed from other operation types.
```json
{
    __style_definition_name__ : {
        "SYNC" : {
            __old_style_name__ : __new_style_name__
        }
    }
}
```
