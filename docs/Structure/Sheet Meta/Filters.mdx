  <TabItem value="filterData" label="Filter Data">
  - The FilterData response will not be included in the document meta or sheet meta.

  <h3>Structure</h3>
    ```json
    {
        "FILTER_CELL_COLOR_LIST": __cell_color_list__,
        "FILTER_TEXT_COLOR_LIST": __text_color_list__,
        "FILTER_LOGICAL_OPERATOR": __logical_operator__,
        "FILTER_DATA_TYPE": __filter_data_type__,
        "FILTER_APPLIED_VALUES": [
            {
                "FILTER_OPERATOR": __filter_operator__,
                "FILTER_SPECIAL_CONDITION": __is_special_condition__,
                "FILTER_SPECIAL_OPERATOR_TYPE": __special_operator_type__,
                "FILTER_SPECIAL_OPERATOR_CATEGORY": __special_operator_category__,
                "FILTER_VALUE": __filter_value__,
                "FILTER_SPECIAL_OPERATOR_ID": __special_operator_id__
            }
        ],
        "FILTER_SELECTED_INDEXES": __selected_index_array__,
        "FILTER_MAX_DATA": __is_max_data_reached__,
        "FILTER_MAX_CELL_COLOR": __is_max_cell_color_reached__,
        "FILTER_MAX_TEXT_COLOR": __is_max_text_color_reached__,
        "APPLIED_FILTER_TYPE": __filter_type__,
        "IS_PICKLIST": __is_picklist__,
        "FILTER_CELL_DATA": __cell_data_array__,
        "PICKLIST_ID" : __picklist_id__
    }
    ```
  <h3>Response</h3>
  <BrowserOnly fallback={<div>Loading JSON...</div>}>
    {() => {
        const JsonViewer = require('@site/src/components/JsonViewer').default;
        return <JsonViewer data={{

                "THEME_COLOR": "TEXT1"
            }
        ],
        "FILTER_LOGICAL_OPERATOR": 1,
        "FILTER_DATA_TYPE": 1,
        "FILTER_APPLIED_VALUES": [
            {
                "FILTER_OPERATOR": "top values",
                "FILTER_SPECIAL_CONDITION": true,
                "FILTER_SPECIAL_OPERATOR_TYPE": "top",
                "FILTER_SPECIAL_OPERATOR_CATEGORY": "items",
                "FILTER_VALUE": "3",
                "FILTER_SPECIAL_OPERATOR_ID": 3

        ],
        "FILTER_SELECTED_INDEXES": [],
        "FILTER_MAX_DATA": false,

<br/>
:::note
- FILTER_SPECIAL_CONDITION, FILTER_SPECIAL_OPERATOR_TYPE, FILTER_SPECIAL_OPERATOR_CATEGORY, and FILTER_SPECIAL_OPERATOR_ID will be present in FILTER_APPLIED_VALUES only when a special operator is applied.
- IS_PICKLIST and PICKLIST_ID will be included when a picklist filter is applied.
:::
